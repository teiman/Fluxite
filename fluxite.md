# Fluxite

Subir proyectos a GitHub es engorroso. Fluxite es una interfaz web local que simplifica todo el proceso.

## Uso

```bash
npm run start
```

Se abre el navegador con la interfaz.

## Funcionalidades

- Leer un directorio y reconocer sus ficheros
- Generar o editar `.gitignore` automáticamente
- Wizard paso a paso para subir un proyecto a GitHub (incluida la creación del repositorio)
- Recordar el token de acceso para no tener que volver a autenticarse

## Objetivo

Fácil y usable.
